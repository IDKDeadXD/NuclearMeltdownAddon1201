# Mekanism Fission Meltdown Rework

A small addon that enhances the meltdown mechanics of the Mekanism **Fission Reactor**.

Instead of a simple explosion, meltdowns now create a large-scale catastrophic event that leaves visible destruction across the landscape.

## **Small Note**: This mod can be put on your server and server only so your clients do not need to install it

## Features

- Massive **radial destruction pattern** when a reactor melts down
- Terrain is blasted outward in dramatic spokes
- Far larger environmental impact than vanilla Mekanism meltdowns
- Creates memorable "nuclear disaster" moments on servers
- Designed to look spectacular from above or in replays

## What Happens During a Meltdown

When a fission reactor reaches meltdown conditions:

1. The core collapses.
2. A massive explosion erupts from the reactor center.
3. Destructive shockwaves carve radial blast paths through the terrain.
4. Fire spreads across the surrounding area.

The result is a huge crater-like disaster zone that clearly shows the power of a reactor failure.

## Compatibility

- Requires **Mekanism** and **Mekanism Generators** 
- Intended for modded survival servers and tech modpacks

## Use Cases

This addon is perfect for:

- Tech modpacks
- SMP servers
- Hardcore engineering gameplay
- Players who want reactor failures to feel **dangerous and cinematic**

## Disclaimer

This mod intentionally makes meltdowns significantly more destructive.  
Build your reactors responsibly.
